# Footer

footer component

## Component Structure

```bash
footer
├─ sass
│  ├─ _default.scss        #The default mobile first CSS
│  ├─ _desktop.scss        #The CSS that would be in effect in desktop
│  ├─ _tablet-l.scss       #The CSS that would be in effect in tablet landscape
│  └─ _tablet-p.scss       #The CSS that would be in effect in tablet portrait
├─ footer.js       #The component javascript functionality
├─ footer.spec.js  #The component javasctipt spec for unit tests
├─ footer.hbs      #The component html template
├─ footer.json     #The component data model
├─ footer.scss     #The component SASS file
└─ README.md     #Readme file
```
