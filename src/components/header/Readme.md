# Header

header component

## Component Structure

```bash
header
├─ sass
│  ├─ _default.scss        #The default mobile first CSS
│  ├─ _desktop.scss        #The CSS that would be in effect in desktop
│  ├─ _tablet-l.scss       #The CSS that would be in effect in tablet landscape
│  └─ _tablet-p.scss       #The CSS that would be in effect in tablet portrait
├─ header.js       #The component javascript functionality
├─ header.spec.js  #The component javasctipt spec for unit tests
├─ header.hbs      #The component html template
├─ header.json     #The component data model
├─ header.scss     #The component SASS file
└─ README.md     #Readme file
```
