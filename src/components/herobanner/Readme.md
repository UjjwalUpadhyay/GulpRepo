# Herobanner

herobanner component

## Component Structure

```bash
herobanner
├─ sass
│  ├─ _default.scss        #The default mobile first CSS
│  ├─ _desktop.scss        #The CSS that would be in effect in desktop
│  ├─ _tablet-l.scss       #The CSS that would be in effect in tablet landscape
│  └─ _tablet-p.scss       #The CSS that would be in effect in tablet portrait
├─ herobanner.js       #The component javascript functionality
├─ herobanner.spec.js  #The component javasctipt spec for unit tests
├─ herobanner.hbs      #The component html template
├─ herobanner.json     #The component data model
├─ herobanner.scss     #The component SASS file
└─ README.md     #Readme file
```
